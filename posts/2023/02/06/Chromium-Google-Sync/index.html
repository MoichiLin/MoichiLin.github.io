Chromium Google Sync

02-06-2023
- -
tutorial-ChromeChromiumGoogleSync

为Chromium开启Google的同步功能

本文来源 凌莞喵

具体前提和起因就不细讲了，
本文具体讲需要做哪些步骤

加入 Chromium dev 用户组

Chromium-dev
Google browser sign-in test account

创建项目

进入Google 开发者中心,创建一个新的项目

进入左边的 API 和服务 > OAuth 同意屏幕先初始化配置自己的应用
User Type 选择外部
接下来的名称啥的都可以随意填写
在测试用户页面，添加自己的账户

添加 API

在库内搜索并添加以下API


Admin SDK API
BigQuery API
BigQuery Storage API
Chrome Web Store API
Cloud Datastore API
Cloud Debugger API
Cloud Logging API
Cloud Monitoring API
Cloud Search API
Cloud SQL
Cloud Storage
Cloud Trace API
Google Cloud APIs
Google Cloud Storage JSON API
Google Drive API
Safe Browsing API
Service Management API
Service Usage API
Time Zone API
VM Migration API

获取 API 密钥

进入凭据页面，首先创建一个 API 密钥
然后创建 OAuth 客户端 ID，应用类型选择桌面应用

添加密钥到环境变量

把以下内容加入自己的key后，写入environment中

sudo vim /etc/environment

export GOOGLE_API_KEY=API 密钥的「键」
export GOOGLE_DEFAULT_CLIENT_ID=客户端 ID
export GOOGLE_DEFAULT_CLIENT_SECRET=客户端密钥

注销再登录即可
